## Mise en ligne GitHub Pages (étapes VS Code)

1. Extraire le ZIP dans un dossier
2. Ouvrir le dossier dans VS Code
3. `Ctrl+Shift+P` → "Git: Initialize Repository"
4. Icône Source Control → message "Initial commit" → ✔ Commit
5. "Publish Branch" → connecte ton compte GitHub
6. Sur GitHub : Settings → Pages → Branch: main → / (root) → Save
7. URL : `https://TON-USERNAME.github.io/portfolio-luc-wu/`
